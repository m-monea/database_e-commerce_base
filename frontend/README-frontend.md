# Frontend demo del progetto database

Questa cartella contiene una piccola interfaccia web statica pensata per mostrare visivamente il progetto database e-commerce.

## File inclusi

- `index.html` -> struttura della pagina
- `styles.css` -> stile grafico
- `app.js` -> dati demo e logica per popolare le sezioni

## Come usarla

1. Apri la cartella `frontend`
2. Fai doppio click su `index.html`
3. La pagina si aprirà nel browser

## Cosa mostra

- dashboard con metriche principali
- elenco utenti
- elenco prodotti
- vista ordini con collegamenti logici tra tabelle
- sezione con spiegazione delle JOIN

## Nota importante

Questa versione è statica: non si collega davvero a un database.
Serve per presentare il progetto in modo visivo.
